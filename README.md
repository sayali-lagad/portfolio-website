# Personal Portfolio Website

## Overview

This is a Full Stack Web Development Internship Project developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB Atlas.

The project consists of a personal portfolio website with a contact form and backend integration.

## Features

- Responsive Portfolio Website
- Navigation Bar
- Hero Section
- Skills Section
- Projects Section
- Contact Form
- Node.js Backend
- Express.js API
- MongoDB Atlas Integration

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

## How to Run the Project

1. Install Node.js
2. Open the project folder
3. Install dependencies:

npm install

4. Start the server:

node server/server.js

5. Open index.html using Live Server in VS Code.
